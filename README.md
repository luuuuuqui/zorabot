## 🔥 Sobre ZORA MD

**ZORA** Era uma base com custo alto e uma qualidade ruim mais agora esta ultra rápido e leve para WhatsApp. Corrigida com foco em performance e facilidade de uso.

**Responsável:** [KimereLs](https://github.com/KimereLs)  
**Repositório:** [Zora-Md](https://github.com/KimereLs/Zora-Md)

---

## ⚙️ Requisitos

Antes de começar, certifique-se de ter:
- **Termux** instalado
- **Conexão de internet** estável
- **Smartphone/Tablet** sendo uma bomba Android 

---

## 📦 Instalação Rápida

### 1️⃣ Instalar Dependências
```bash
pkg upgrade -y && pkg update -y && pkg install nodejs -y && pkg install nodejs-lts -y && pkg install ffmpeg -y && pkg install wget -y && pkg install git -y
```
> Digite **`y`** quando solicitado

### 2️⃣ Liberar Armazenamento
```bash
termux-setup-storage
```

### 3️⃣ Clonar Repositório
```bash
cd /sdcard && git clone https://github.com/KimereLs/Zora-Md
```

### 4️⃣ Instalar Dependências do Bot
```bash
cd Zora-Md && npm install
```

---

## 🚀 Iniciar o Bot

Escolha uma das opções abaixo:

```bash
npm start
```

**OU**

```bash
sh zora.sh
```

## 🛠️ Troubleshooting

**Bot não inicia?**
- Verifique se o Node.js está instalado
- Rode `npm start` novamente

**Erro de conexão?**
- Reinicie o Termux
- Verifique sua conexão de internet
